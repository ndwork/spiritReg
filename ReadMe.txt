
The main routine is run_reconSpiritRegCS.  This is the routine that should be run to generate results with the data provided.

The ankle data provided with this code was first made available with the following publication:
Dwork, Nicholas, Jeremy W. Gordon, and Erin K. Englund. "Accelerated parallel magnetic resonance imaging with compressed sensing using structured sparsity." Journal of Medical Imaging 11, no. 3 (2024): 033504-033504.

This code requires PISCO, which can be gotten from here: https://mr.usc.edu/download/pisco/
